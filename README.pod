=head1 NAME

Mensa::Dresden - Perl interface to receive offerings of Dresden's canteens

=head1 INSTALLATION

To install this module type the following:

  perl Makefile.PL
  make
  make test
  make install

=head1 DEPENDENCIES

The modules and the script require these other modules and libraries:

  XML::LibXML
  XML::LibXSLT
  YAML::Tiny
